# 使用VuePress搭建的简单个人博客

VuePress本是一个用于写技术文档的框架，它的blog功能也没有特别完善。但是它可以编译Markdown文件为前端页面的特点让我第一时间选择了它。与此同时也可以写Vue插件来自定义样式，简直不能再棒。

# 为什么有了这个blog?

  • 将学过的东西记录下来
  • 培养个人定期写作习惯
  • 方便回顾个人所解决过的问题
  • 少量盈利

# 搭建博客前对博客的要求：

  • 搭建简单，快速
  • 方便维护
  • Markdown编辑文本
  • SEO友好
  • 前端使用Vue或者React
